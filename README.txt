1) Настройки окон независимы друг от друга
2) Таймер не выполняет сохранение новых файлов(Безымянные), только те, которые уже сущесвтуют на компьютере, 0 секунд - выключить автосохранение.
3) Если вызывается что-то открыть в текущей вкладке, но в выборе нового файла нажали "Отмена" - текущая вкладка все равно удалится.
4) Если вкладок нет, то "в текущей" == "в новой"
5) Сохранение настроек:
     а) Настройки окна берутся из последней закрытой формы.
     б) Новый файлы, которые пользователь не захотел сохранять не будут повторны открыты.
     в) При открытии настройки каждого текстового поля соответсвуют настройкам последнего сохраненного файла(самой правой вкладки)
6) Журналирование будет создано в папке Logs, которая лежит вместе с sln, чтобы не возникало проблема если уже есть такая папка на компьютере.
7) Журналирование не применимо к новым файлам, только при наличии созданного файла на копьютере можно включить журналиирование, 
    которое будет сохранять версии начиная с текущей.
8) Журналирование, как и другие настройки сохраняются для всех файлов при повторном открытии, если у последнего закрытого файла оно было включено
9) Файлы не гигантские! не нужно перепонять стек плз
10) "Цвет текста в поле ввода" - это как и изменить цвет выбранного цвет, так и переключить цвет ввода
11) "Настройка шрифта" - это как изменить шрифт ввода, так и изменить шрифт ввода
12) !!! Сохранение шрифта и цвета текста в поле ввода работает следующим образом:
        а) если последний закрытый файл последней закрытой формы был путстым, то установленные шрифт и цвет текста применяться ко всем файлам
            при открытии(у них такой будет шрифт и цвет ввода). На весь текст!
        б) если последний закрытый файл последней закрытой формы был непустым, то шрифт и цвет сохраняются в сответсвии с первым элементом
            последнего закрытого файла и применяется ко всем вновь открывшимся. На весь текст!
